# Labolatoria-WdSM
Repozytorium – Wprowadzenie do Systemów Mobilnych
Opis:
To repozytorium zawiera materiały oraz rozwiązania z laboratoriów realizowanych w ramach przedmiotu Wprowadzenie do Systemów Mobilnych na Uniwersytecie w Siedlcach.

W ramach zajęć analizowane są zagadnienia związane z działaniem sieci komórkowych. Każde laboratorium znajduje się w osobnym katalogu i zawiera opis teoretyczny w README oraz wykonane zadania.

Struktura repozytorium
Każdy folder lab_XX odpowiada kolejnemu laboratorium i zawiera:
opis zadania, analizę teoretyczną oraz wykonane obliczenia lub implementację.
